# Feastie - Color Palette Reference

## 메인 컬러 분석

| 역할 | 색상명 | Hex | 용도 |
|------|--------|-----|------|
| **Primary** | 딥 퍼플 | `#4A1A8A` | 로고, 헤드라인, 주요 텍스트 |
| **Secondary** | 핫 핑크 | `#FF69B4` | CTA 버튼, 배경 섹션, 장식 요소 |
| **Accent** | 레드-오렌지 | `#E53620` | 배지, 강조 배경, 포인트 요소 |

## 전체 컬러 팔레트

### Background Colors
```css
--bg-primary:    #FFF8F0;  /* 크림 화이트 - 메인 배경 */
--bg-pink:       #FF69B4;  /* 핫 핑크 - 섹션 배경 */
--bg-blue:       #5BB8E8;  /* 스카이 블루 - 장식 원형 배경 */
--bg-red:        #E53620;  /* 레드-오렌지 - 이미지 프레임 배경 */
```

### Text Colors
```css
--text-primary:   #4A1A8A;  /* 딥 퍼플 - 헤드라인, 주요 텍스트 */
--text-dark:      #1A1A1A;  /* 블랙 - 본문 텍스트 */
--text-inverse:   #FFFFFF;  /* 화이트 - 어두운 배경 위 텍스트 */
--text-sub:       #6B5B7B;  /* 라이트 퍼플 - 보조 텍스트 */
```

### Accent & Highlight Colors
```css
--accent-lime:    #C8E655;  /* 라임 그린 - "SIP" 텍스트 배경 */
--accent-yellow:  #FFB830;  /* 옐로우 - "SNACK" 텍스트 */
--accent-pink:    #F0A0D0;  /* 라이트 핑크 - "FESTIVAL" 배경 */
--accent-marker:  #FF69B4;  /* 핑크 십자(+) 장식 마커 */
```

### Border / Outline
```css
--border-primary: #4A1A8A;  /* 퍼플 아웃라인, 점선 데코 */
--border-dotted:  #4A1A8A;  /* 점선 구분선 */
--border-light:   #E0D8E8;  /* 연한 퍼플 보더 */
```

### Interactive Elements
```css
--btn-primary-bg:    #FF69B4;  /* GET TICKETS 버튼 배경 */
--btn-primary-text:  #4A1A8A;  /* 버튼 텍스트 */
--btn-hover-bg:      #FF4DA6;  /* 버튼 호버 상태 */
--link-color:        #4A1A8A;  /* 네비게이션 링크 */
--link-hover:        #E53620;  /* 링크 호버 */
```

## 컬러 사용 비율

| 색상 | Hex | 비율 | 사용 목적 |
|------|-----|------|-----------|
| 크림 화이트 | `#FFF8F0` | ~35% | 메인 배경, 여백, 콘텐츠 영역 |
| 딥 퍼플 | `#4A1A8A` | ~25% | 로고, 타이틀, 텍스트, 아웃라인 |
| 핫 핑크 | `#FF69B4` | ~20% | 배경 섹션, CTA 버튼, 장식 요소 |
| 스카이 블루 | `#5BB8E8` | ~10% | 이미지 배경 원형 도형 |
| 레드-오렌지 | `#E53620` | ~5% | 배지 배경, 원형 사진 프레임 |
| 라임 그린 | `#C8E655` | ~3% | 텍스트 하이라이트 배경 |
| 옐로우 | `#FFB830` | ~2% | 텍스트 강조 색상 |

## 컬러 사용 가이드

### 1. 타이포그래피
- **헤드라인(H1~H2)**: `#4A1A8A` 딥 퍼플 사용. 볼드 웨이트와 함께 브랜드 아이덴티티를 강조
- **본문(Body)**: `#1A1A1A` 블랙 사용. 크림 배경 위에서 높은 가독성 확보
- **보조 텍스트**: `#6B5B7B` 라이트 퍼플로 계층 구분
- **반전 텍스트**: 핑크/블루 배경 위에서는 `#FFFFFF` 화이트 사용

### 2. 버튼 & CTA
- **Primary CTA**: 배경 `#FF69B4` + 텍스트 `#4A1A8A` (퍼플과 핑크의 대비)
- **Secondary CTA**: 배경 투명 + 보더 `#4A1A8A` + 텍스트 `#4A1A8A`
- **호버 상태**: Primary는 `#FF4DA6`으로 약간 진하게, Secondary는 배경 `#F5E6FF` 추가

### 3. 배경 섹션
- **기본 섹션**: `#FFF8F0` 크림 화이트
- **강조 섹션**: `#FF69B4` 핑크 배경 + 화이트 텍스트
- **교차 섹션**: 크림 ↔ 핑크 ↔ 크림으로 시각적 리듬감 형성
- **장식 도형**: `#5BB8E8` 블루, `#E53620` 레드를 원형으로 활용

### 4. 장식 & 그래픽 요소
- **점선 데코**: `#4A1A8A` 퍼플 점선으로 섹션 구분
- **십자(+) 마커**: `#FF69B4` 핑크로 티커/배너 장식
- **배지/라벨**: 다중 컬러 레이어링 (라임 → 레드 → 핑크 순서)

### 5. 이미지 & 미디어
- **이미지 프레임**: 원형 도형에 `#E53620` 레드 또는 `#5BB8E8` 블루 배경
- **오버레이**: 사용하지 않음 — 이미지는 선명하게 노출
- **그림자**: 최소한으로 사용, 도형 자체의 컬러 대비에 의존

## 반응형 고려사항

### 라이트 모드 (기본)
```css
:root {
  --bg-primary:     #FFF8F0;
  --text-primary:   #4A1A8A;
  --text-body:      #1A1A1A;
  --accent-pink:    #FF69B4;
  --accent-blue:    #5BB8E8;
  --accent-red:     #E53620;
  --accent-lime:    #C8E655;
  --accent-yellow:  #FFB830;
}
```

### 다크 모드
```css
[data-theme="dark"] {
  --bg-primary:     #1A0A2E;  /* 딥 다크 퍼플 */
  --text-primary:   #D4B8FF;  /* 라이트 퍼플 */
  --text-body:      #E8E0F0;  /* 연한 라벤더 화이트 */
  --accent-pink:    #FF85C8;  /* 약간 밝은 핑크 (접근성) */
  --accent-blue:    #6DC8F5;  /* 밝은 블루 */
  --accent-red:     #FF5A45;  /* 밝은 레드 */
  --accent-lime:    #D4F070;  /* 밝은 라임 */
  --accent-yellow:  #FFCA55;  /* 밝은 옐로우 */
}
```

### 모바일 (< 600px)
- 핑크 배경 섹션의 패딩 축소 (`spacing(4)` → `spacing(2)`)
- 장식 원형 도형 크기 50%로 축소 또는 숨김 처리
- 십자 마커, 점선 데코 등 장식 요소 간소화
- CTA 버튼 풀 와이드(`width: 100%`)로 확장

### 태블릿 (600px ~ 1200px)
- 2컬럼 레이아웃에서 핑크/블루 배경 도형 비율 유지
- 이미지 프레임 원형 도형은 가로 폭의 40%로 제한
- 네비게이션은 햄버거 메뉴로 전환

### 접근성 (WCAG 2.1 AA)
| 조합 | 대비율 | 통과 여부 |
|------|--------|-----------|
| 퍼플 `#4A1A8A` on 크림 `#FFF8F0` | 9.2:1 | AA/AAA 통과 |
| 블랙 `#1A1A1A` on 크림 `#FFF8F0` | 15.8:1 | AA/AAA 통과 |
| 화이트 `#FFFFFF` on 핑크 `#FF69B4` | 3.1:1 | 대형 텍스트만 AA 통과 |
| 퍼플 `#4A1A8A` on 핑크 `#FF69B4` | 3.0:1 | 대형 텍스트만 AA 통과 |

> **주의**: 핑크 배경 위 텍스트는 18px 이상 볼드 또는 24px 이상에서만 사용 권장.
> 소형 텍스트에는 `#2E0E5E`(더 진한 퍼플)로 대체하여 대비율 확보.

## 디자인 특징

- **분위기**: 활기차고 축제적. 퍼플과 핑크의 조합이 트렌디하면서도 친근한 인상
- **스타일**: 레트로-팝. 90~2000년대 영감의 볼드 타이포그래피와 원색 조합
- **레이아웃**: 유기적인 원형 도형과 불규칙한 배치로 수공예적 감성
- **대비 전략**: 크림 배경 위 딥 퍼플 텍스트로 가독성 확보, 컬러풀한 도형으로 시각적 흥미 유지
