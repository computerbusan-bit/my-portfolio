# Design System

## Color Palette

### Primary Colors
- **Primary**: `#1976d2` (Blue)
- **Primary Light**: `#42a5f5`
- **Primary Dark**: `#1565c0`

### Secondary Colors
- **Secondary**: `#dc004e` (Pink)
- **Secondary Light**: `#ff5983`
- **Secondary Dark**: `#9a0036`

### Neutral Colors
- **Background**: `#ffffff`
- **Surface**: `#f5f5f5`
- **Text Primary**: `rgba(0, 0, 0, 0.87)`
- **Text Secondary**: `rgba(0, 0, 0, 0.6)`

## Typography

- **Font Family**: `"Roboto", "Helvetica", "Arial", sans-serif`
- **H1**: 2.125rem / fontWeight 500
- **H2**: 1.5rem / fontWeight 500
- **H3**: 1.25rem / fontWeight 500
- **Body1**: 1rem / fontWeight 400
- **Body2**: 0.875rem / fontWeight 400

## Spacing

- Base unit: `8px`
- Use MUI `theme.spacing(n)` for consistent spacing

## Components

- Use MUI components as the base building blocks
- Customize via `theme.components` overrides when needed
- Maintain consistent border-radius: `4px` (default MUI)
